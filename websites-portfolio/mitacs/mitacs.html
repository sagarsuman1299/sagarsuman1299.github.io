Ecomania Details

1. Conducted thorough research into the issue of improper garbage disposal.
2. Created "Ecomania," an educational game teaching proper waste disposal through interactive gameplay.
3. Designed engaging mini-games to subtly convey essential lessons in responsible waste management.
4. Developed interactive features - Burger Making, Quiz Play, Garbage Runner, Asset Purchase, and user registrations and Leaderboard using Unity3D, ExpressJS,MongoDB.
