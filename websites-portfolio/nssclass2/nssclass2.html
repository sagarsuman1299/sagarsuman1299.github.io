Task 1:

1.Bitpack application/Website :
The project was related to building a trading platform where web3 users can list and trade their NFTs without going through too many procedures. They simply have to signup using their personal details and then buy or sell their NFTs using our payment gateway build through razorpay. Our primary objective was to build a website using flutter but since flutter is a cross platform framework, we made use of packages catering to all platforms and responsive to fit into multiple platform that are website,android and ios.
Along with that we have tested the entire applications manually through excel to make it production ready.

  • State Management Library: GetX
  • Tech Stacks: Flutter,RestAPI

    • Task 2:

    2.Bitpack navigator :
    The project was related to building a platform where web3 users can see their popular NFTs and details related to them.
    The platform was build on the top of NextJS to facilitate server-side-rendering. Along with that we have used Typescript as it is faster and errors are detected during compile time.

  • Tech Stacks: Typescript,NextJS,RestAPI

    • Task 3:

    3. API hosting :
    The provided apis were to be hosted on some platform so that anyone in our development team can use the latest api for their usage. So, we have developed a react application which converted our Postman json into swagger UI directly and the entire website was hosted on netlify. CI/CD was applied to this project so that any update pushed to github gets directly updated to the website in real time.

  • Tech Stacks: Swagger,React,Netlify
